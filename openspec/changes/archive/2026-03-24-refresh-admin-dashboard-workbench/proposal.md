## Why

管理员工作台已经接入真实登录上下文、权限上下文和系统配置链路，但当前页面的信息层次还不够完整，视觉表达也没有把“排课系统主控台”的价值充分呈现出来。现在左侧菜单已经提供了稳定的 `权限管理` 入口，工作台中的“权限管理联调入口”卡片变成了重复导航，应该删除并让首页承担更清晰的总览与状态展示职责。

## What Changes

- 重做管理员工作台首页的信息架构，强化“排课控制台”而不是“开发联调页”的定位。
- 删除工作台中的“权限管理联调入口”卡片，避免与左侧菜单重复。
- 优化首页视觉设计，保留当前学院档案馆式基调，但提升版式、层次、数据卡片与状态模块的完成度。
- 将首页内容重心改为系统主链状态、排课域概览、配置状态与后续工作引导，而不是单一权限联调提示。
- 补充工作台页面的前端验收标准，确保优化后在桌面端与移动端都可用。

## Capabilities

### New Capabilities
- `admin-dashboard-workbench`: 定义管理员工作台的首页布局、信息模块、视觉层次和导航约束。

## Impact

- 受影响代码主要为 `UI/coursearrange/src/views/dashboard/DashboardView.vue`。
- 可能需要微调与首页相关的样式结构，但不涉及后端 API 变更。
- 不改变左侧菜单、RBAC、路由守卫或数据库菜单结构，只优化前端首页呈现与交互。
